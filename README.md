# MoMA Moment

🏆 **Best Overall Winner — MoMA's First Ever Hackathon (in collaboration with NYU SPS)**

## Competition Prompt
> How might MoMA reimagine the onsite visitor experience, through design, technology, or communication, to help visitors more easily navigate the museum, discover art that resonates with them, and feel more connected before, during, and after their visit?

## Overview
MoMA Moment is an innovative project designed to transform the visitor experience at the Museum of Modern Art. Built during MoMA's inaugural hackathon, this project leverages technology and thoughtful design to help visitors:
- Seamlessly navigate the museum
- Discover artworks that resonate with their interests
- Foster deeper connections before, during, and after their visit

## Features
- Interactive gallery and camera switch functionality
- Personalized art discovery
- Enhanced navigation tools
- Integration with Supabase for backend support

## Project Structure
- `index.html`, `gallery.html`, `complete.html`: Main web pages
- `camera-switch.js`: Camera and interaction logic
- `supabase-setup.sql`: Database setup for Supabase integration
- `assets/`: Contains images and other media assets

## Getting Started
1. Clone this repository.
2. Open `index.html` in your browser to explore the experience.
3. Review `supabase-setup.sql` for backend setup instructions if you wish to deploy your own instance.
